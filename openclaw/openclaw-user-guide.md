## Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/ssaahhil832/ssaahhil832.github.io.git

# Navigate to the project
cd openclaw

# Install dependencies
npm install
```

### Running the Application

```bash
# Development mode
npm run dev

# Production build
npm run build
```
